# Data-Science-Assignment-2# 🎬 Movie Data Analysis (Assignment 2)

## 📊 Overview
This project analyzes a dataset of ~3000 movies to extract useful insights related to profitability, actors, directors, and languages.  
The goal is to support decision-making for a movie producer using data-driven analysis.

---

## ❓ Questions Solved
1. Identified the movie with the highest profit along with its cast, producer, and director  
2. Determined which language gives the highest average ROI  
3. Extracted all unique genres present in the dataset  
4. Found top 3 producers based on average ROI  
5. Identified the actor with the most movies and analyzed their performance  
6. Analyzed top directors and their most frequently chosen actors  

---

## ⚙️ Approach
- Cleaned and processed the dataset  
- Created new features:
  - **Profit = Revenue – Budget**
  - **ROI = Profit / Budget**
- Extracted nested data (director, genres, producers)  
- Used grouping and aggregation for insights  

---

## 🛠️ Tools Used
- Python  
- Pandas  
- NumPy  
- AST (for parsing structured data)  

---

## 📈 Key Insights
- Profitability depends on both budget and revenue efficiency  
- Some languages provide better ROI despite smaller scale  
- Certain producers and directors consistently perform well  
- Actor-director collaborations are common in successful movies
